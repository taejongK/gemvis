# Gemvis - 팀 공유용 방향성 정리

## 한 줄 정의
**Gemvis는 온디바이스에서 개인 데이터를 자동으로 정리하고, 사람·장소·시간·프로젝트 단위의 로컬 지식그래프로 연결해, 대화형으로 다시 찾을 수 있게 해주는 프라이버시 우선 AI 비서입니다.**

---

## 우리가 풀고 있는 문제
사람들은 정보를 저장하지 못해서 잃어버리는 것이 아니라, **너무 많이 저장해서 나중에 다시 찾지 못하게 됩니다.**

메모, 스크린샷, 사진, 음성메모, PDF, 다운로드 파일 등은 계속 쌓이지만,
그것들이 정리되지 않고 서로 연결되지 않기 때문에 결국 디바이스는 기억 저장소가 아니라 잡동사니 창고가 됩니다.

Gemvis는 이 문제를 해결합니다.

---

## 제품 정체성
Gemvis는 단순한 파일 정리 앱이 아닙니다.

이 제품은:
- 개인 디바이스 안에서
- 새로 저장되는 디지털 흔적을 이해하고
- 안전하게 분류하고 정리하고
- 사람, 장소, 시간, 프로젝트, 이벤트 단위로 연결하고
- 나중에는 자연어로 다시 찾아주는
**온디바이스 개인 지식그래프 비서**입니다.

즉, 핵심은 파일 관리가 아니라
**개인 데이터 → 구조화된 기억/지식 → 대화형 회수**입니다.

---

## 왜 Gemma 4여야 하는가
이 프로젝트는 단순히 LLM을 붙였기 때문에 의미가 있는 것이 아니라,
**Gemma 4의 온디바이스, 멀티모달, tool calling 특성이 핵심 가치와 직접 연결되기 때문에 의미가 있습니다.**

### 핵심 포인트
1. **오프라인 동작 가능성**
   - 인터넷이 없어도 동작할 수 있어야 함
   - 개인 비서로서 항상 접근 가능해야 함

2. **프라이버시 절대 보장**
   - 사진, 메모, 음성, 문서 같은 민감한 개인 데이터를 클라우드에 보내지 않아도 됨
   - 데이터가 디바이스를 떠나지 않는 것이 큰 경쟁력

3. **온디바이스 성능 개선 체감**
   - 기존 온디바이스 모델로는 품질이 부족했던 분류, 정리, 관계 연결, 멀티모달 이해를 더 높은 수준으로 제공해야 함

4. **멀티모달 + tool calling + reasoning 결합**
   - 사진, 텍스트, 음성, 문서를 이해하고
   - 실제 정리 액션을 제안/실행하고
   - 나중에는 자연어로 다시 찾아주는 구조가 가능해야 함

---

## 지식그래프가 중요한 이유
이 제품에서 지식그래프는 단순 장식이 아니라 핵심 차별점입니다.

### 지식그래프 없이 가능한 것
- 파일 분류
- 태그
- 폴더 정리
- 키워드 검색

### 지식그래프가 있을 때 가능한 것
- 여러 소스의 데이터 연결
- 시간축 추론
- 관계 추론
- 설명 가능한 회수

예를 들면 이런 질문이 가능해집니다.
- “지난달 김과장이랑 같이 간 식당 어디였지?”
- “그때 찍은 사진이랑 음성메모도 보여줘”
- “해커톤 관련 아이디어들 묶어서 보여줘”

즉 Gemvis는 단순 검색이 아니라,
**흩어진 개인 데이터를 관계형 기억으로 재구성하는 제품**이어야 합니다.

---

## 소구 포인트
### 1. 프라이버시
- 개인 데이터가 디바이스 밖으로 나가지 않음
- 민감한 정보도 안심하고 다룰 수 있음

### 2. 디지털 과부하 해소
- 저장 과잉으로 생기는 인지부하를 줄임
- 찾지 못하는 문제를 해결함

### 3. 자동 정리 + 대화형 회수
- 새 파일을 알아서 이해하고 정리함
- 나중에 자연어로 다시 찾게 해줌

### 4. 기억 중심 구조화
- 단순 폴더 정리가 아니라
- 사람, 장소, 프로젝트, 이벤트 단위로 기억을 엮음

### 5. 신뢰 가능한 반자동
- 모든 걸 자동으로 처리하지 않음
- low-risk는 자동, sentimental/ambiguous는 review-first

## MVP 방향
범위를 넓히면 실패하므로, MVP는 반드시 파일 기반 로컬 데이터에 집중합니다.

### MVP 입력 소스
- 메모/텍스트
- 스크린샷/사진
- 음성메모
- PDF
- 다운로드 폴더

### MVP에서 하는 것
- 새 파일 감지
- 분류
- 요약
- 태그
- suggested folder 제안
- auto-safe / review-first 분기
- 대화형 검색
- 경량 지식그래프 연결

### MVP에서 하지 않는 것
- 모든 SNS 연동
- 소비/금융/메시지/헬스 데이터 전체 통합
- 동영상 깊은 분석
- 완전한 모바일 지원
- 완전 자동 삭제
- 무거운 그래프 DB 도입
